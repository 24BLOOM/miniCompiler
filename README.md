# miniComplier





## 实现功能：

①　使用未定义的变量

②　调用未定义的函数

③　同一作用域，变量、数组、结构体、函数的重复定义

④　函数调用的时实参和形参不匹配

⑤　函数调用时参数个数不匹配

⑥　对非函数名采用非函数调用形式访问

⑦　对非数组变量采用下标变量的形式访问

⑧　数组变量的下标不是整型表达式

⑨　数组变量下标位附属

⑩　数组访问越界

⑪　赋值号左边不是左值表达式

⑫　赋值语句需要左值

⑬　对非左值表达式进行自增、自减运算

⑭　对结构体变量进行自增、自减运算

⑮　类型不匹配

⑯　函数返回值类型与函数定义不一致

⑰　函数没有返回(当函数返回值不是void时）

⑱　除0报错

⑲　没有main函数
